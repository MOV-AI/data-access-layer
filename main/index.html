Data Access Layer

The Data Access Layer (DAL) is an abstraction layer Library for accessing data. The data may be read/written (pending permission) from/to the followin data sources:

  • GIT

  • DB (Redis)

  • File system

The Data Service provides:

  1. Validation of user permissions before accessing the data

  2. Enforcing the appropriate JSON schema when writing data

  3. Validating compatibility between a document versions

Usage

The DAL exposes APIs for reading and writing the data and is used by MOV.AI platform components. It’s a python package by message-server, backend and flow-initiator services.
